PexelApp

**Overview**
App uses PexelApi: the best free stock photos, royalty free images & videos shared by creators.

**Tech Stack**
- Android SDK
- Jetpack Compose
- Jetpack Navigation
- Room
- Retrofit
- Dagger/Hilt
- Coroutines + Flow
- Paging3

**Before usage**
To run the application, you should first add your API key to "local.properties" as "API_KEY"
